# Google Sheet Employee Form

Use these files inside a Google Sheet:

1. Open your Google Sheet.
2. Go to **Extensions > Apps Script**.
3. Create or replace `Code.gs` with the contents of `Code.gs`.
4. Click **Save**.
5. Reload the Google Sheet.
6. Use **Employee Tools > Open Data Entry** to open the sheet-based form.

Submissions are saved in a sheet tab named `201 file`.

Reload the Google Sheet after saving the script. In the `201 file` sheet, type a name or ID number in cell `B2` to search. Clear `B2` to show all records again.

In the `Data Entry` sheet, fill out the form and either check the `Submit` checkbox or use **Employee Tools > Submit Data Entry**.
